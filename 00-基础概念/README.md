# 00 - 基础概念

> 看完 3Blue1Brown 视频之后，读这里。从零建立对 AI 模型的完整理解。

## 为什么需要这个模块

本知识库的 01-07 章涉及大量前沿技术（MoE、MLA、GRPO、DualPipe 等），默认你已经理解基础概念。如果你直接看会觉得——

- "Attention 到底在干什么？"
- "模型怎么从一堆随机数变成能对话的？"
- "训练具体是个什么过程？"
- "为什么大家都在用 Transformer？之前用什么？"

**这个模块就是来解决这些问题的。**

## 三篇文档

| 文档 | 内容 | 阅读时间 | 读完你能回答 |
|------|------|---------|-------------|
| [模型演化史](./模型演化史.md) | 从感知机到 GPT-4o 的完整演化链 | ~30 分钟 | 为什么是 Transformer？为什么是 Decoder-Only？ |
| [Transformer 从零理解](./Transformer从零理解.md) | 从输入到输出，每个组件拆解 | ~40 分钟 | Attention 怎么算的？Q/K/V 是什么？GQA 是什么？ |
| [训练与推理全流程](./训练与推理全流程.md) | 训练怎么做、推理怎么跑 | ~35 分钟 | 什么是反向传播？为什么需要并行？KV Cache 是什么？ |

