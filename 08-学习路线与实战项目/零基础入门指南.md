## 二、第二步：本地跑模型（1-2 天）

> 看完视频后，立刻在自己电脑上跑一个模型。**亲手触摸 AI** 比任何文档都有效。

### 最快路径：Ollama（10 分钟上手）

```bash
# 1. 下载安装 Ollama
# Windows: https://ollama.com/download/windows
# 安装后打开终端

# 2. 跑第一个模型（自动下载，约 5GB）
ollama run qwen3:8b

# 3. 开始对话！试试这些：
#    - "用简单的话解释什么是 Transformer"
#    - "写一个 Python 快速排序"
#    - "1+1=? 请一步一步思考"（观察 thinking 模式）

# 4. 再试一个推理模型，感受区别
ollama run deepseek-r1:7b
#    - "一个房间有3个开关控制隔壁3盏灯，你只能进隔壁一次，如何确定对应关系？"
#    - 观察它的长思维链（<think>...</think>）
```

### 推荐体验的模型

| 模型 | 命令 | 大小 | 体验重点 |
|------|------|------|---------|
| **Qwen3-8B** | `ollama run qwen3:8b` | ~5GB | 中文最强，支持思考模式 |
| **DeepSeek-R1 7B** | `ollama run deepseek-r1:7b` | ~5GB | 看推理模型怎么"思考" |
| **Llama 3.1 8B** | `ollama run llama3.1:8b` | ~5GB | 英文能力，Meta 出品 |
| **Phi-3 Mini** | `ollama run phi3` | ~2GB | 微软小模型，极快 |

### 对比实验（建立体感）

给不同模型同一个问题，感受差异：

```
数学题：  "证明 √2 是无理数"
编程题：  "用 Python 实现一个简单的 HTTP 服务器"
创意题：  "写一首关于人工智能的七言绝句"
推理题：  "小明比小红大3岁，小红比小华小2岁，小华今年10岁，小明几岁？"
```

## 六、常见问题

### "我需要学数学吗？"

- Agent 开发方向：**不需要**。API 调用和框架使用不涉及数学
- 理解模型原理：需要基础线性代数和概率论（看 3Blue1Brown 就够了）
- 训练模型：需要更多数学（但这不是你的重点）

### "我需要什么硬件？"

- Agent 开发：**任何电脑都行**（用 API）
- 本地跑模型：至少 16GB RAM（CPU 推理）或 RTX 3060+ (GPU 推理)
- 微调模型：RTX 3060 12GB 可以用 QLoRA 微调 7B 模型
- 训练模型：RTX 4090+ 或用云 GPU

### "英文论文看不懂怎么办？"

- 本文档就是中文版的技术解读，覆盖了最重要的论文
- 用 Claude/GPT 翻译和解释论文段落
- 先看博客解读（Sebastian Raschka, Cameron Wolfe），再看原论文

### "Agent 开发需要懂模型训练吗？"

- **不需要**。就像用 Web 框架不需要懂编译器一样
- 但了解基本原理（什么是 Token、什么是上下文窗口、什么是 Temperature）会让你写出更好的 Agent
- 本文档的 01-02 章提供了足够的背景知识
