# 08 - 学习路线与实战项目

> 零基础到 Agent 开发者的完整学习路线。理论看文档建立认知，动手跑项目获得体感。

## 你的情况

- AI/ML 零基础，想建立全景认知
- 有编程基础（Python/TypeScript）
- 有家用电脑（RTX 3060/4060, 16-32GB RAM）
- **重点方向：Agent 开发**（模型训练了解即可）
- 目标：能独立开发 Agent 应用，理解 AI 技术全貌

## 总览

```
学习路线（约 4-5 个月）

第一阶段：建立直觉（1-2 周）
  看视频/科普 → 本地跑模型 → 建立"AI 能做什么"的体感

第二阶段：全景扫描（2-3 周）
  快速浏览本文档 01-07 章 → 不求全懂，建立框架

第三阶段：Agent 开发入门（2-3 周）
  API 调用 → 工具调用 → MCP Server → 第一个 Agent

第四阶段：Agent 开发进阶（5-7 周）
  LangGraph → 多 Agent → RAG → Deep Research

第五阶段：本地 AI 体验（穿插进行）
  图像生成 → 语音合成 → 微调 → 视频生成

第六阶段：挑战项目（可选，6-10 周）
  Agent SDK → Computer Use → RL 训练 → 生产部署
```

## 学习原则

1. **先跑起来再理解**：不要试图先看懂所有数学公式再动手
2. **体感 > 理论**：本地跑一次 DeepSeek-R1 比读 10 页论文更有效
3. **Agent 优先**：模型训练了解原理即可，Agent 开发才是你的发力点
4. **项目驱动**：每学一个概念就做一个小项目验证
5. **不要贪多**：宁可把一个项目做透，也不要浅尝辄止 10 个
