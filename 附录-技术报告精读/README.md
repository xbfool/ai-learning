# 附录 - 技术报告精读

> 三篇定义当代大模型工程范式的核心论文，逐章精读。

---

## 定位

本目录收录的是**深度精读**级别的技术报告解读，每篇 700-900 行，覆盖论文的核心技术细节和关键数字。

如果你只想快速了解一个模型的要点，请看 [05-开源模型Tech-Report](../05-开源模型Tech-Report/)（200-500 行概要）。如果你想深入理解"为什么这样设计"和"具体怎么实现的"，请看本目录。

## 三篇精读

| 报告 | 论文 | 为什么必读 | 精读重点 |
|------|------|-----------|---------|
| [Llama 3 精读](./Llama-3-精读.md) | The Llama 3 Herd of Models (arxiv 2407.21783) | **最全面的全栈 LLM 工程参考** — 从数据到架构到训练到安全，每个环节都有详尽文档 | 数据工程、4D 并行、Scaling Laws、后训练流水线 |
| [DeepSeek-V3 精读](./DeepSeek-V3-精读.md) | DeepSeek-V3 Technical Report (arxiv 2412.19437) | **MoE + FP8 + MLA 工程典范** — $5.57M 训练出 GPT-4o 级别模型，定义 2025 技术范式 | MoE 负载均衡、MLA 压缩、FP8 训练、DualPipe |
| [OLMo 3 精读](./OLMo-3-精读.md) | OLMo 3 (arxiv 2512.13961) | **完全开源的预训练流水线** — 唯一开源了数据+代码+检查点的模型项目 | Dolma 3 数据集、Swarm 优化、RLVR 后训练 |

## 阅读建议

```
路线 A（工程导向）：Llama 3 → DeepSeek-V3 → OLMo 3
  先建立全栈认知，再看极致工程优化，最后看完全可复现流程

路线 B（成本导向）：DeepSeek-V3 → OLMo 3 → Llama 3
  先看 $5.57M 的奇迹怎么做到，再看开源替代方案

路线 C（研究导向）：OLMo 3 → Llama 3 → DeepSeek-V3
  从完全透明的项目入手，理解训练全貌，再看工业级实现
```
