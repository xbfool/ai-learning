# 09 - 软件架构内功修炼

> AI 能写代码，但不能设计系统。架构能力是 AI 时代工程师最重要的护城河。
> 本章整理架构设计的核心书单、学习路线和实践笔记，结合日常项目边工作边修炼。

## 为什么要学架构

```
AI 时代的能力分层：

  能被 AI 替代的        ← 写 CRUD、套模板、调 API
  ──────────────────
  AI 辅助但人做决策      ← 代码审查、技术选型、性能诊断
  ──────────────────
  AI 无法替代的          ← 系统设计、权衡取舍、跨团队协作
                           ↑ 这就是架构能力
```

架构不是画框图，而是**在约束条件下做出可逆的（或代价可控的）技术决策**。
