# 图像生成

> 从 Stable Diffusion 到 FLUX.2，图像生成经历了架构范式的根本转变。

## 一、架构演进时间线

```
2022: Stable Diffusion 1.x → U-Net + DDPM + CLIP
2023: SDXL → 更大 U-Net + 双文本编码器
2024: SD3 → MMDiT (Transformer 替代 U-Net) + Flow Matching
2024: FLUX.1 → 双流 DiT + Rectified Flow
2025: FLUX.2 → 32B 参数 DiT + VLM 耦合
2025: GPT-4o → 纯自回归图像生成 ← 范式转变
```

## 二、核心架构

### DiT (Diffusion Transformer)
- **论文**：Peebles & Xie (2023)
- 用 Vision Transformer 替代 U-Net 做扩散去噪
- **优势**：遵循 Scaling Laws，模型越大效果越好
- **劣势**：计算量大于 U-Net
- 被 FLUX, SD3, Sora 等所有 SOTA 模型采用

### Flow Matching / Rectified Flow
- **核心**：学习从噪声到数据的直线路径 (OT path)
- **替代**：传统 DDPM 的弯曲扩散路径
- **优势**：
  - 轨迹更直 → 需要更少的采样步骤
  - 更容易蒸馏到 1-4 步生成
  - 训练目标更简单 (velocity matching)
- 被 FLUX, SD3, 大部分视频模型使用

### 自回归图像生成 (AR)
- **核心**：将图像 token 化，用 LLM 方式逐 token 生成
- **代表**：GPT-4o Images, Emu3, LlamaGen, VAR
- **优势**：
  - 与文本生成统一架构
  - 指令遵循能力强
  - 文字渲染准确
  - 多轮编辑自然
- **劣势**：高频细节不如扩散模型

### 混合架构
- **Transfusion** (Meta)：文本自回归 + 图像扩散，共享一个 Transformer
- **GLM-Image** (智谱)：自回归骨干 + 扩散解码器，中文/多语言排版出色
- **Show-o**：自回归文本 + 离散扩散图像

## 三、当前 SOTA 模型

### FLUX.2 (Black Forest Labs, 2025.11)
- [官网](https://bfl.ai/models/flux-2)
- **架构**：32B 参数 Latent Flow Matching Transformer
  - 耦合 Mistral-3 24B VLM（视觉语言模型）
  - Rectified Flow Transformer 做图像潜空间操作
- **变体**：
  - FLUX.2 [pro]：最高质量，API 使用
  - FLUX.2 [flex]：开发者版，细粒度控制
  - FLUX.2 [dev]：32B 开源权重，支持本地部署
- **核心能力**：
  - 支持最多 10 张参考图
  - 角色一致性、产品外观、视觉风格保持
  - 文字渲染准确
- **地位**：开源图像生成的当前标杆

### Stable Diffusion 3.5 (Stability AI)
- MMDiT (Multi-Modal DiT)：文本和图像 token 在同一 Transformer 中联合注意力
- 三个文本编码器：CLIP ViT-L + OpenCLIP ViT-bigG + T5-XXL
- Rectified Flow 训练
- Medium / Large / Turbo 三个版本

### GPT-4o Native Image Generation (OpenAI, 2025.03)
- **范式突破**：不是扩散模型，而是自回归 token 生成
- 优势：
  - 极强的指令遵循
  - 精确的文字渲染
  - 自然的多轮编辑
  - 统一理解和生成
- 激发了对统一多模态模型的广泛兴趣

### HiDream-I1 (2025.04)
- 17B 参数
- 在关键基准上超越 SDXL、DALL-E 3 和 FLUX.1
- 开源新星

### 其他值得关注
- **Recraft V4**：设计和矢量图形领域强势
- **Ideogram 2.0+**：文字渲染特别出色
- **Midjourney v7**：美学质量最高（闭源）
- **Kolors** (快手)：开源，中文友好

## 四、训练方法论

### 数据处理
1. **合成标注** (Synthetic Captioning)：用 VLM 对训练图像重新生成详细描述
   - DALL-E 3 首创此方法
   - 显著提升 prompt 遵循度
2. **渐进式训练**：低分辨率 → 高分辨率，逐步提升
3. **多分辨率/多比例训练**：适应不同输出尺寸

### 对齐 (Alignment)
- **Diffusion-DPO**：将 DPO 应用到扩散模型
- **奖励模型微调**：用人类偏好奖励模型指导扩散模型
- **CFG (Classifier-Free Guidance)**：仍是标准技术，但部分模型（FLUX schnell）已不需要

### 加速
- **蒸馏**：将多步扩散蒸馏为 1-4 步
- **一致性模型 (Consistency Models)**：直接预测干净图像
- **对抗蒸馏**：用 GAN loss 辅助蒸馏
